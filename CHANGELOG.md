## [0.1.0] - 2026-03-12

### Added

- 🚚 Initial `puzzletree` package, CLI entrypoint, and project tooling.
- ✨ Tile image splitting and reconstruction CLI commands with staged progress reporting.
- ✨ Core reconstruction graph, rendering, frame packing, export controls, and candidate inspection reports.
- ✅ Reconstruction fixture coverage and supporting runtime dependencies.
- 📝 Project documentation, including README visuals and PyPI publishing aliases.

### Changed

- ♻️ Improved reconstruction defaults, packing behavior, and layout handling.
- 🔧 Cleaned up repository configuration and local artifact handling.

### Fixed

- 🐛 Corrected reconstruction layout gaps and animation packing issues.
- 🩹 Fixed README asset linking for the reconstruction GIF.
- 🩹 Ensured CI runs the main quality checks.
